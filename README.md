🖩 Modern Calculator

A simple, modern calculator built with HTML, CSS, and JavaScript.



🚀 Features

✅ Basic arithmetic operations: addition, subtraction, multiplication, and division.

✅ Clear display (AC) and delete last entry (DE) functionality.

✅ Responsive design with a sleek and modern UI.



📂 Files

📌 index.html - The main HTML file that contains the structure of the calculator.

📌 style.css - The CSS file that styles the calculator.

📌 Inline JavaScript in index.html - Handles the calculator's functionality.



📌 Usage

1️⃣ Clone the repository or download the files.

2️⃣ Open index.html in your web browser.



🏗 Structure

📜 index.html

🛠 Contains the HTML structure of the calculator, including the display and buttons.

🎨 style.css

🎨 Contains the CSS styles for the calculator, including layout, colors, and button styles.
